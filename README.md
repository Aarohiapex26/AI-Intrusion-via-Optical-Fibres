> **Where Fibres Listen and AI Think**  
A web‑based system that simulates and analyzes signals from optical fibre sensors to detect footsteps, vehicles, animals, and insects. It combines **AI classification**, **real‑time alerts**, and **map visualization** with a comparison tool for signal analysis.

---

## 🚀 Features
- **Signal Simulation**: Generate synthetic signals for footsteps, vehicles, heavy vehicles, animals, and insects.
- **AI Classification**: TensorFlow.js model trained to classify activity types based on extracted features.
- **Real‑Time Alerts**: Visual + audio alerts for suspicious activity with severity levels (low, medium, high).
- **Location Mapping**: Interactive Leaflet map showing your actual location and detected activity markers nearby.
- **Activity Comparison**: Select two activities from dropdowns, view their signals side‑by‑side, and compare features in a table.

---

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Charts**: Chart.js
- **AI Model**: TensorFlow.js
- **Mapping**: Leaflet + OpenStreetMap
- **Alerts**: HTML5 Audio + dynamic DOM updates
